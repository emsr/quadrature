\doxysubsection{Detailed Description}
Header file for jacobi library. 

The jacobi library implements Jacobi polynomials and Gauss-\/\+Jacobi quadrature related functions. This library was developed to be used in high order finite element methods (Spectral Element/hp methods. The interfaces were inspired by the software Polylib (\href{http://www.nektar.info}{\texttt{ http\+://www.\+nektar.\+info}}).

The main sources used to develop this library where\+:

(1) Abramowitz, Milton and Stegun, Irene (editors); \char`\"{}\+Handbook of Mathematical functions\char`\"{}, Dover Publications 1965.

(2) Karniadakis, George Em and Sherwin, Spencer; \char`\"{}\+Spectral/hp Element Methods for Computational     Fluid Dynamics\char`\"{}, Oxford Science Publications, 2nd edition, 2005.
