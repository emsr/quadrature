◆ increment_curr_index()

template<typename _Tp , typename _RetTp >
bool __gnu_cxx::integration_workspace< _Tp, _RetTp >::increment_curr_index ( )

Increase the heap start point until the current segment has a smaller depth than the current maximum depth. After each increment rebuild the heap from the new start point so the new start point is the largest error.

Usage: In the caller the smallest interval (at max depth) has the largest error. Before bisecting decrease the sum of the errors over the larger intervals (error_over_large_intervals) and perform extrapolation.
