Calculates the derivative of the nth order Jacobi polynomial at an array of np points given by the vector x. This function needs two workspaces of size np to do its calculation. If the first one (ws1) is NULL, the function will allocate the necessary memory (using malloc) to do the calculation

Parameters
npNumber of points where the polynomial will be calculated
xA vector of length np containing the points where the polynomials should be calculated. .
nOrder of the Jacobi polynomial whose derivative should be calculated
result_arrayAn array of length np where the result will be stored
a parameter of Jacobi polynomial
b parameter of Jacobi polynomial
wsA pointer to a block of memory 2*np doubles long to be used as workspace. If it is NULL, memory will be allocated using malloc and released at the end
Returns
Error code defined in gsl_errno.h or GSL_SUCCESS if everything was fine
