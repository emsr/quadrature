121 /**
122  * Fast Fourier Transform
123  *
124  * Discrete Fourier Transform can be regarded as evaluating a
125  * polynomial of degree N-1 on the powers
126  * @f$ \omega^0, \omega, \omega^2, ..., \omega^(N-1) @f$
127  * where @f$ \omega @f$ is the Nth root of unity.
128  *
129  * Given a polynomial of even degree
130  * @f[
131  * p(t) = a_0 + a_1 t + a_2 t^2 + ...
132  * @f]
133  * we find:
134  * @f[
135  * p(t) = a_0 + a_2 t^2 + a_4 t^4 + ... + t(a_1 + a_3 t^2 + ...)
136  * = q_{even}(t^2) + t q_{odd} (t^2)
137  * @f]
138  * where q_e and q_o are polynomials formed with the even and odd
139  * coefficients of p(t). Thus, we get
140  *
141  * p(1) = q_{even}(1) + \omega^0 q_{odd}(1)
142  * p(\omega) = q_{even}(\omega^2) + \omega^1 q_{odd}(\omega^2)
143  * p(\omega^2) = q_{even}(\omega^4) + \omega^2 q_{odd}(\omega^4)
144  * ...
145  *
146  * Note how on the RHS the Fourier transforms of q_e and q_o appear. Thus:
147  */
